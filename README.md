# HTML and CSS course #
Learn to build your very first web pages with this four hour course on HTML and CSS.

### You'll learn ###
*What is HTML and CSS?
*Terminology and syntax
*HTML tags
*File naming and project setup
*Anchors
*Attributes
*__strong__ and *emphasis*
*CSS basics
*Classes and Ids
*Internal & external CSS
*Borders
*The box model
*Margin & padding
*Lists
*Images
*Centering elements
*Styling your page
